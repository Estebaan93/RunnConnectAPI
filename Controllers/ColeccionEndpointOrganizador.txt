COLECCION PERFIL ORGANIZADOR
Post Login - http://localhost:5213/api

ORGANIZADOR (4):
1. POST  /Evento/ - Crea un evento, el perfil del orga debe estar completo para crear el evento
Authorizacion: token bearer (organizador) evalua que sea token de organizador
Content-Type: application/json
  Body:
    json
      {
        "nombre": "Maratón San Luis 2025",
        "descripcion": "Carrera de 10K y 5K por las calles de San Luis",
        "fechaHora": "2026-01-15T08:00:00",
        "lugar": "Plaza Pringles, San Luis",
        "cupoTotal": 500,
        "urlPronosticoClima": "https://www.weather.com/sanluisargentina",
        "datosPago": "CBU: 0000003100012345678901 - Alias: RUNNERS.SL - Titular: Runners Club San Luis S.A"
      }

2. GET /Evento/MisEventos - Ver mis eventos (todos los estados) de un organizador
Authorizacion: token bearer (organizador) evalua que sea token de organizador
Content-Type: application/json
Request: 
  json 
    {
  "total": 1,
  "eventos": [
    {
      "idEvento": 1,
      "nombre": "Maratón San Luis 2025",
      "fechaHora": "2026-01-15T08:00:00",
      "lugar": "Plaza Pringles, San Luis",
      "estado": "publicado",
      "cupoTotal": 500,
      "inscriptosActuales": 0,
      "cantidadCategorias": 0,
      "nombreOrganizador": "" --- Revisar no me trae el nombre y/o razon social organizador
    }
  ]
}


3. PUT /Evento/{id} - Actualizar un evento - Si esta cancelado y/o finalizado no se puede editar
Authorizacion: token bearer (organizador) evalua que sea token de organizador
Content-Type: application/json
Body:
  json
    {
      "nombre": "Maratón San Luis 2025 - ACTUALIZADO",
      "descripcion": "Carrera de 10K y 5K - Descripción actualizada",

4. PUT /Evento/{id}/CambiarEstado - Cambiar estado a (publicado/cancelado/finalizado)
Authorizacion: token bearer (organizador) evalua que sea token de organizador
Content-Type: application/json

    }
